COMMENT TESTER:

1. Ouvre 2 TERMINAUX

TERMINAL 1 (Frontend):
$ cd frontend
$ npm install
$ npm run dev
→ http://localhost:5173

TERMINAL 2 (Backend):
$ cd backend
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan serve
→ http://localhost:8000/api

2. Ouvre http://localhost:5173
3. Clique "Sign up"
4. Crée un compte
5. Tu es connecté ✅
