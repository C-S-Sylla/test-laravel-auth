🔐 LES 6 ENDPOINTS API:

POST   /api/auth/logout          → Se déconnecter (Protégé)
PUT    /api/auth/profile         → Modifier profil (Protégé)
DELETE /api/auth/profile         → Supprimer compte (Protégé)

❌ PROBLÈMES COURANTS:

❌ "Port 5173 occupé?"
✅ cd frontend && npm run dev -- --port 3000

❌ "Port 8000 occupé?"
✅ php artisan serve --port 9000

❌ "Erreur base de données?"
✅ cd backend && php artisan migrate:fresh && php artisan migrate

❌ "Erreur CORS?"
✅ Vérifier backend/config/cors.php

❌ "npm: command not found?"
✅ Installer Node.js depuis nodejs.org

❌ "composer: command not found?"
✅ Installer Composer depuis getcomposer.org
