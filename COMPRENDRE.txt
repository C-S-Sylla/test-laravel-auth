🚀 POUR LANCER L'APPLICATION:

Ouvre 2 TERMINAUX (2 consoles/cmd)

═══════════════════════════════════════════════════════════════════

TERMINAL 1 - REACT (Frontend)
────────────────────────────────────

Tape ceci:
$ cd frontend
$ npm install
$ npm run dev

Ensuite ça va afficher:
  ➜  Local:   http://localhost:5173/

C'est bon! Ne ferme pas ce terminal.

═══════════════════════════════════════════════════════════════════

TERMINAL 2 - LARAVEL (Backend)
────────────────────────────────────

Tape ceci:
$ cd backend
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan serve

Ensuite ça va afficher:
  INFO  Server running on [http://127.0.0.1:8000]

C'est bon! Ne ferme pas ce terminal.

═══════════════════════════════════════════════════════════════════

OUVERTURE DE L'APP
────────────────────

1. Ouvre ton navigateur
2. Va à: http://localhost:5173
3. Clique sur: "Don't have an account? Sign up"
4. Remplis le formulaire d'inscription
5. Tu es connecté! ✅

═══════════════════════════════════════════════════════════════════

QU'EST-CE QUI SE PASSE?

Quand tu remplis le formulaire:

USER → React (frontend/)
       ↓
       Envoie données à http://localhost:8000/api/auth/register
       ↓
       Laravel (backend/) reçoit
       ↓
       Valide les données
       ↓
       Crée l'utilisateur en base de données
       ↓
       Génère un token
       ↓
       Renvoie le token au React
       ↓
       React sauvegarde le token
       ↓
       React te connecte automatiquement
       ↓
       Tu vois le dashboard ✅
