## 🚀 LANCER L'APPLICATION

### TERMINAL 1 - FRONTEND (React)
```
cd frontend
npm install
npm run dev
```
→ S'ouvre à: http://localhost:5173

### TERMINAL 2 - BACKEND (Laravel)
```
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
→ API à: http://localhost:8000/api

## ❌ PROBLÈMES COURANTS

PROBLÈME: "npm: command not found"
SOLUTION: Installer Node.js depuis nodejs.org

PROBLÈME: "php: command not found"
SOLUTION: Installer PHP depuis php.net

PROBLÈME: "composer: command not found"
SOLUTION: Installer Composer depuis getcomposer.org

PROBLÈME: Port 5173 occupé
SOLUTION: cd frontend && npm run dev -- --port 3000

PROBLÈME: Port 8000 occupé
SOLUTION: php artisan serve --port 9000

PROBLÈME: Database error
SOLUTION: 
  cd backend
  php artisan migrate:fresh
  php artisan migrate

PROBLÈME: CORS errors
SOLUTION: Vérifier backend/config/cors.php
